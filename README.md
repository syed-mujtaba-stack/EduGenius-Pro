## ğŸ› ï¸ Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/syed-mujtaba-stack/EduGenius-Pro.git
   cd EduGenius-Pro
   ```

2. **Install frontend dependencies**
   ```bash
   cd client
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   ```
   http://localhost:3000
   ```
